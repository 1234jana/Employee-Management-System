# 👨‍💼 Employee Management System (Django)

## 📌 Project Overview

The Employee Management System is a web-based application developed using Django that enables efficient management of employee records. It provides full CRUD functionality, allowing users to add, view, update, and delete employee details through a clean and user-friendly interface.

---

## 🛠️ Technologies Used

* 🐍 Python
* 🌐 Django
* 🗄️ SQLite (default database)
* 🎨 HTML, CSS, Bootstrap

---

## ✨ Features

* ➕ Add new employee records
* 📋 View employee list
* 🔍 View detailed employee information
* ✏️ Update employee details
* ❌ Delete employee records with confirmation
* 🎨 Responsive UI using Bootstrap

---

## 📂 Project Structure

* `employees/` – Django app (models, views, forms)
* `templates/` – HTML templates
* `db.sqlite3` – Database file
* `manage.py` – Django management script

---

## ▶️ How to Run the Project

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

   ```bash
   pip install django
   ```
4. Apply migrations:

   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```
5. Run the server:

   ```bash
   python manage.py runserver
   ```
6. Open in browser:

   ```
   http://127.0.0.1:8000/
   ```
